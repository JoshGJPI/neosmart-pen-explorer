NeoSmartpen Data Explorer

This application lets you explore and backup data from your NeoSmartpen without removing or erasing it from the pen.

Important: This application runs locally in your browser and does not send any data to external servers. All data stays on your computer.

Browser Requirements

This application requires a browser that supports the Web Bluetooth API:

  • Google Chrome
  • Microsoft Edge
Note: This application will not work in Firefox, Safari, or Internet Explorer as they do not support the Web Bluetooth API.

How to Use

  1. Open index.html in Chrome or Edge.
  2. Click the Connect Pen button to establish a Bluetooth connection with your NeoSmartpen.
  3. Select your pen from the Bluetooth device list that appears.
  4. Once connected, the application will display a list of notes stored on the pen.
  5. Click on a note to view its pages.
  6. Click on a page to view its data.
  7. Use the dropdown menu to switch between raw JSON data and visual preview.
  8. Use the Download Data button to export data for the selected page.
  9. Use the Backup All Data button to export all data from the pen.

Features

  • Non-destructive Reading: Views pen data without deleting anything.
  • Visual Previews: See your handwritten notes visualized on screen.
  • Data Export: Save pen data as JSON files for backup or analysis.
  • Offline Access: Works completely offline, no internet required.

Local File Limitations

Due to browser security restrictions when running local HTML files:

  • The application uses a fallback implementation for demonstration purposes.
  • For full functionality, consider using a local web server or hosting the files.
Tip: If you have Node.js installed, you can easily run a local server:
  1. Open a command prompt in this directory
  2. Run: npx http-server .
  3. Open http://localhost:8080 in your browser

Troubleshooting

  • Pen not connecting: Make sure your pen is turned on and in Bluetooth pairing mode.
  • Bluetooth permission denied: You need to grant Bluetooth permission in your browser.
  • Visual preview not working: Make sure Fabric.js is loading correctly (check browser console).
